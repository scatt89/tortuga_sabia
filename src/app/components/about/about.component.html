La Tortuga Sabia es un proyecto didáctico realizado por M. Da Silva Garcia.

Esta aplicación web consiste en un visualizador de consejos, normalmente dados por nuestra amiga La Tortuga Sabia. Aparte de disfrutar de los maravillosos consejos de la tortuga, los usuarios también pueden publicar sus propios consejos, de forma que todos los usuarios se benefician de la sabiduria colectiva.

Tienes todo el código del proyecto disponible en mi repositorio de GitHub, eres libre de consultar y compartir lo que quieras.

Para realizar estre proyecto se ha usado Angular2 y Typescript en el front. En el back Java 8, Spring MVC, Spring Boot, Hibernate, MySQL y ElasticSearch.
